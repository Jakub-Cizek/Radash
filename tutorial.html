V tomto ppad se jedn o slitinu Nb2TaTi3VZr2.
Syntaxe tdy Material je nsledujc:
radash.Material(name,elements,concentrations,kwargs)
parametry:
name : string
oznaen materilu
elements : tuple of strings
chemick prvky z kterch je slitina sloena
concentrations : array of floats
koncentrace jednotlivch prvk
kwargs
units: atomic, weight, default = atomic
jednotky, v kterch jsou zadny koncentrace jednotlivch prvk (implicitn se pedpokldaj atomrn koncentrace).
Koncentrace jednotlivch prvk nemus bt normovan (zde jsme zadali, e jedna "molekula" slitiny obsahuje 2 atomy Nb, 1 atom Ta, 3 atomy Ti, 1 atom V a 2 atomy Zr). Program provede normalizaci a vype atomrn koncentrace jednotlivch prvk.
Pokud jsou koncentrace zadny v hmotnostnch procentech s paramterem units = weight program je pepot do atomovch procent a dle pouv atomrn koncentrace.


K oznaen prvk pouvme standardn chemick znaky, prvn psmeno velk (nap. Nb, V atd.) Pi definovn slitiny je mon pout jakkoliv chemick prvek.
Pokud pouijeme znaku neexistujcho prvku, program ohls chybu, nap.
In[9]:
hea2=radash.Material('CzNb2TaTi3VZr2',['Cz','Nb','Ta','Ti','V','Zr'],[3,2,1,3,1,2],units='atomic') 

error: unknown element symbol Cz

    Radian pokozen NRT dpa zpsoben neutrony s rznou energi je mon vypotat pomoc metody
    get_NRT_dpa_spectrum(fluence,thickness,kwargs)
    parametry:
    fluence : float
    celkov fluence (poet neutron na cm2)
    thickness : float
    tlouka ozaovanho vzorku (mm) kwargs :
    spectrum = file name
    parametr spectrum unouje zadat jmno souboru obsahujcho energetick spektrum neutron.
    Spektrum neutron je oekvno ve dvou sloupcovm formtu E[i] I[i], kde E[i] je energie v jednotkch MeV a I[i] je relativn zastoupen neutron s touto energi ve spektru. Zadan spektrum nemus bt nijak normovan, program sm provede renormalizaci spektra na jednotkovou plochu.
    plot = True, False, default = True
    Pokud je tento pepna True, tak program vykres energetick spektrum neutron.
    V tomto konkrtnm ppad jsme pouili spektrum neutron experimentlnho reaktoru LVR 15 v Centru vzkumu e, kde byly ozaovny vzorky slitin s vysokou entropi v rmci tohoto projektu TAR. Toto spektrum je uloeno s souboru nazvanm neutron_energy_spectrum_LVR15.txt. Energetick spektrum jadernho reaktoru obsahuje velmi vysok pspvek termlnch neutron. Fluence neutron je vtinou udvna pro energetick neutrony s energi vy ne 0.5 MeV. Frakce takovch energetickch neutron ve spektru reaktoru LVR 15 je $5.47\times 10^{-2}$. Proto je fluence korigovna touto frakc tak, aby zadan hodnota fluence odpovdala fluenci energetickch neutron $E > 0.5$ MeV.

    In[29]:
    fast_neutron_fraction=5.47e-2 #frakce neutron E > 0.5 MeV ve spektrum reaktoru LVR 15
hea1.get_NRT_dpa_spectrum(fluence/fast_neutron_fraction,thickness,spectrum='neutron_energy_spectrum_LVR15.txt',plot=True)

    spectrum file: neutron_energy_spectrum_LVR15.txt

    fluence: 18.282 x 1e16 1/cm2,  NRT dpa: 0.660
