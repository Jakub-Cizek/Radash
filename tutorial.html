Ukázka použití software RADASH na výpočet radiačního poškození slitin s vysokou entropií

Nejdříve importujeme knihovnu radash, která obsahuje definici třídy objektů Material a metody pro práci s objekty této třídy

In [3]:
import radash

Dále importujeme také knihovnu pro numerické výpočty numpy a knihovnu pro tvorbu grafů matplotlib.pyplot
Obě tyto knihovny jsou standardní součástí distribuce Anaconda

In [5]:
import numpy as np
import matplotlib.pyplot as plt

Nyní můžeme definovat nějakou slitinu. Vytvoříme tedy nový objekt třídy Material s názvem hea1

In [7]:
hea1=radash.Material('Nb2TaTi3VZr2',['Nb','Ta','Ti','V','Zr'],[2,1,3,1,2],units='atomic')

number of constituents: 5

V tomto případě se jedná o slitinu Nb2TaTi3VZr2.
Syntaxe třídy Material je následující:
  radash.Material(name,elements,concentrations,kwargs)
  parametry:
  name : string
  označení materiálu
  elements : tuple of strings
  chemické prvky z kterých je slitina složena
  concentrations : array of floats
  koncentrace jednotlivých prvků
  kwargs
  units: atomic, weight, default = atomic
  jednotky, v kterých jsou zadány koncentrace jednotlivých prvků (implicitně se předpokládají atomární koncentrace).
Koncentrace jednotlivých prvků nemusí být normované (zde jsme zadali, že jedna "molekula" slitiny obsahuje 2 atomy Nb, 1 atom Ta, 3 atomy Ti, 1 atom V a 2 atomy Zr). Program provede normalizaci a vypíše atomární koncentrace jednotlivých prvků.
Pokud jsou koncentrace zadány v hmotnostních procentech s paramterem units = weight program je přepočítá do atomových procent a dále používá atomární koncentrace.


K označení prvků používáme standardní chemické značky, první písmeno velké (např. Nb, V atd.) Při definování slitiny je možné použít jakýkoliv chemický prvek.
Pokud použijeme značku neexistujícího prvku, program ohlásí chybu, např.
In [9]:
hea2=radash.Material('CzNb2TaTi3VZr2',['Cz','Nb','Ta','Ti','V','Zr'],[3,2,1,3,1,2],units='atomic') 

error: unknown element symbol Cz

Program umožňuje vypočítat různé fyzikální parametry slitiny, kterou jsme vytvořili.
Tyto parametry jsou vypočítané jako složením vážený průměr hodnot jednotlivých prvků, ze kterých se slitina skládá. Vypočtené parametry jsou užitečné pro hledání optimálního chemického složení slitiny pro danou aplikaci.

    Za srovnání vypočtené hodnoty NRT dpa a arc dpa je patrné, že atermální zotavení vedlo k rekombinaci 81% Frenkelových párů vytvořených PKA.


    Při výpočtu arc dpa poškození používá program funkci $\xi (E)$ vyjadřující pravděpodobnost "přežití" Frenkelových párů definovanou vztahem (3). Konstanty $b$, $c$ této funkce jsou opět vypočítány jako složením vážený průměr hodnot pro prvky tvořící slitinu.
    Hodnoty konstant $b$ a $c$ pro jednotlivé prvky jsou uloženy v souboru arc_dpa_constants.txt.
    Funkci $\xi (E)$ pro danou slitinu je možné zobrazit metodou get_arc_dpa_constants(**kwargs) s parametrem plot = True

    Radiační poškození NRT dpa způsobené neutrony s různou energií je možné vypočítat pomocí metody
    get_NRT_dpa_spectrum(fluence,thickness,kwargs)
      parametry:
      fluence : float
      celková fluence (počet neutronů na cm2)
      thickness : float
      tloušťka ozařovaného vzorku (mm)   kwargs :
      spectrum = file name
      parametr spectrum unožňuje zadat jméno souboru obsahujícího energetické spektrum neutronů.
    Spektrum neutronů je očekáváno ve dvou sloupcovém formátu E[i] I[i], kde E[i] je energie v jednotkách MeV a I[i] je relativní zastoupení neutronů s touto energií ve spektru. Zadané spektrum nemusí být nijak normované, program sám provede renormalizaci spektra na jednotkovou plochu.
      plot = True, False, default = True
      Pokud je tento přepínač True, tak program vykresí energetické spektrum neutronů.
    V tomto konkrétním případě jsme použili spektrum neutronů experimentálního reaktoru LVR 15 v Centru výzkumu Řež, kde byly ozařovány vzorky slitin s vysokou entropií v rámci tohoto projektu TAČR. Toto spektrum je uloženo s souboru nazvaném neutron_energy_spectrum_LVR15.txt. Energetické spektrum jaderného reaktoru obsahuje velmi vysoký příspěvek termálních neutronů. Fluence neutronů je většinou udávána pro energetické neutrony s energií vyšší než 0.5 MeV. Frakce takových energetických neutronů ve spektru reaktoru LVR 15 je $5.47\times 10^{-2}$. Proto je fluence korigována touto frakcí tak, aby zadaná hodnota fluence odpovídala fluenci energetických neutronů $E > 0.5$ MeV.

    In [29]:
    fast_neutron_fraction=5.47e-2 #frakce neutronů E > 0.5 MeV ve spektrum reaktoru LVR 15
hea1.get_NRT_dpa_spectrum(fluence/fast_neutron_fraction,thickness,spectrum='neutron_energy_spectrum_LVR15.txt',plot=True)

    spectrum file: neutron_energy_spectrum_LVR15.txt

    fluence: 18.282 x 1e16 1/cm2,  NRT dpa: 0.660
